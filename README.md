# 🤖 Бот "50 фраз native speakers"

Telegram-тренажёр для отработки 50 разговорных фраз на английском. Уровень A1–A2.

---

## 📁 Структура файлов

```
bot.py           — главный файл, логика бота
database.py      — работа с SQLite
phrases.py       — контент: 50 фраз + 4 задания к каждой
config.py        — токен и настройки
requirements.txt — зависимости
```

---

## 🚀 Быстрый старт

### 1. Получи токен бота
Напиши [@BotFather](https://t.me/BotFather) в Telegram:
- `/newbot` → придумай имя → получи токен вида `7123456789:AAF...`

### 2. Вставь токен в config.py
```python
BOT_TOKEN = "7123456789:AAF..."
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти бота
```bash
python bot.py
```

---

## ☁️ Деплой на Railway (бесплатно)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. Создай новый проект → **Deploy from GitHub**
3. Загрузи все файлы в репозиторий на GitHub
4. В Railway: **Variables** → добавь `BOT_TOKEN = твой_токен`
5. В `config.py` замени строку на:
   ```python
   import os
   BOT_TOKEN = os.getenv("BOT_TOKEN")
   ```
6. Railway сам найдёт `requirements.txt` и запустит `bot.py`

---

## 🔔 Пуш-уведомления

По умолчанию бот шлёт фразу дня в **10:00 по Москве**.
Чтобы изменить время — отредактируй `config.py`:
```python
PUSH_HOUR = 10    # час (0–23)
PUSH_MINUTE = 0   # минуты
```

---

## 📊 Как работает режим "Слабые места"

Бот считает точность ответов по каждой фразе отдельно.
После нажатия **🔁 Повторить слабые** — выбирает 10 фраз
с наименьшим процентом правильных ответов.
